## What Changed from v2

v2 had gamification-heavy content (XP, levels, badges, ENGAGE buttons).
v3 replaces all gamification with mastery-insight metrics:

| Element Removed | Replacement |
|---|---|
| Global XP | Today's Focus (timer) |
| SysAdmin Lvl.07 | Mastery Index (%) |
| ENGAGE buttons | Clickable concept cards |
| ACQUIRED ASSETS badges | Contribution Heatmap |
| XP Earned (focus) | Concepts Reviewed |
